## 📌 What this app does

PrismML-Bonsai-vs-Qwen3.5-Benchmark helps you review and compare benchmark results from two model setups: Bonsai and Qwen3.5. It gives you a simple way to open the app, load the data, and inspect results on a Windows PC.

Use it when you want to check:

- which model scored higher
- how each run performed
- how results changed across tests
- which setup gave the better outcome

The app is made for direct use. You do not need to set up a coding tool or work in a terminal.

## ❓ Common issues

### The app does not open

Try these steps:

1. Right-click the file.
2. Choose Run as administrator.
3. Check that the file is fully extracted if it came from a zip.
4. Download the latest release again.

### The file will not load

Check that:

- the file is not damaged
- the file format is supported
- the data has clear columns
- the file is not open in another program

### The screen looks empty

This can happen if:

- no data is loaded yet
- the file has no benchmark rows
- the wrong sheet or tab is selected
- the data file uses a layout the app cannot read

### The app runs slowly

Try:

- closing other apps
- using a smaller dataset
- using less browser or video activity in the background
- checking that your PC has enough free memory
